[RUNNING MAN DEMO](https://running-man.azureedge.net/)

## Run locally

### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]
- Install Typescript

### Usage

To build and run:
```bash
frakas serve
```
This with compile typescript files, start a backend run time and basic web server to host game.
